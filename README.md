# 🌸 JLPT AI Trainer — 日本語の勉強  
### *Master Japanese Vocabulary with AI-powered Practice*

> 🎯 An interactive, gamified web application designed to help learners crack the JLPT (N5–N3) with confidence.

---

## 🗾 About the Project

✨ *JLPT AI Trainer* is not just a quiz app — it's a **smart learning experience** inspired by Japanese aesthetics and modern web technologies.

Built for learners who want:
- 🎌 Structured JLPT preparation  
- 🧠 Engaging & memory-friendly quizzes  
- 🌏 A visually immersive Japanese-themed UI  

Whether you're starting your journey or leveling up, this app transforms **learning into an experience**.

---

## 🌟 Highlights

- 🎯 **Level-based Learning** — Practice vocabulary from N5 → N3  
- 🧠 **Smart Quiz System** — Multiple-choice questions with instant feedback  
- 🏆 **Gamification** — Earn points & track your performance  
- 🔐 **Secure Authentication** — OTP-based email verification  
- 👩‍💻 **Admin Dashboard** — Monitor users & performance insights  
- 🌸 **Aesthetic UI** — Inspired by Japanese seasons  
- 📱 **Responsive Design** — Works seamlessly across devices

## 🛠 Tech Stack

💻 Backend        → Flask (Python)
🗄 Database       → SQLite / MySQL
🎨 Frontend       → HTML, CSS, JavaScript
🔐 Authentication → OTP via Gmail SMTP


## 🚀 Quick Start

### 🔧 Setup in 4 Steps

# 1. Clone the repository
git clone https://github.com/sandrasaju/NIHONGO_BENKYOU.git

# 2. Navigate into the project
cd NIHONGO_BENKYOU/jlpt-ai-trainer

# 3. Install dependencies
pip install flask werkzeug

# 4. Run the app
python app.py

🌐 Open in browser:  
👉 http://127.0.0.1:5000
